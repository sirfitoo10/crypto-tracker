# ₿ Crypto Tracker

A simple and modern cryptocurrency tracker that displays real-time prices using a public API.

---

## ✨ Features
- 📈 Real-time cryptocurrency prices
- 🔝 Top 10 coins display
- ⚡ Fast and lightweight
- 🧩 Easy to understand and extend

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- CoinGecko API
